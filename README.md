# FileReaderExample

A simple Java program to read and display the contents of a text file using `BufferedReader`.

## How to Run

1. Compile:
   ```bash
   javac FileReaderExample.java
   ```

2. Run:
   ```bash
   java FileReaderExample
   ```

3. Enter the path to the file or use `sample.txt` for testing.
